# Data loader

This is a simple script for creating preprocessed pytorch datasets out of commonly used datasets.

## Available datasets
 - MNIST
 - TUANDROMD
 - BreastCancer
 - letter
 - ColorectalCarcinoma
 - ColorectalCarcinomaCLR
 - Bioresponse
 - EyeMovements
 - Cleveland
 - Ionosphere
 - Libras
 - Lympography
